# 🎯 Guess The Number

A fun and interactive number guessing game built using **HTML, CSS, and JavaScript** where players try to guess the correct number with limited attempts, score tracking, and smart hints.

## 🚀 Live Demo
https://ayush2614.github.io/guess-the-number/

---

## ✨ Features

✅ Random number generation  
✅ Smart hints (Too High / Too Low)  
✅ Score system  
✅ Lives counter  
✅ Restart game option  
✅ Win / Lose game states  
✅ Responsive UI  
✅ Interactive gameplay  
✅ Beginner-friendly JavaScript project  

---

- CSS3
- JavaScript (Vanilla JS)

## 🎮 How to Play

1. Enter a number between the given range.
2. Click **Guess** button.
3. Get hints:
   - 📈 Too High
   - 📉 Too Low
4. Guess the correct number before lives run out.
5. Restart and play again.
